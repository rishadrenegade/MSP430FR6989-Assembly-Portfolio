# TI MSP430FR6989 Basics with Assembly

This repository contains a series of embedded systems exercises and projects developed on the Texas Instruments MSP430FR6989 microcontroller using Assembly language. The goal of these projects is to build a strong understanding of low-level programming, microcontroller architecture, and direct hardware control.

## Topics Covered
- GPIO Configuration
- LED Control
- Push-Button Interfacing
- Timer Modules
- Interrupt Service Routines (ISR)
- Low-Power Modes
- Register-Level Programming
- Memory-Mapped I/O
- Embedded Systems Fundamentals

## Hardware
- TI MSP430FR6989 LaunchPad Development Kit

## Skills Demonstrated
- Embedded Systems Development
- Assembly Language Programming
- Microcontroller Programming
- Debugging and Testing
- Hardware-Software Integration
- Real-Time Event Handling
